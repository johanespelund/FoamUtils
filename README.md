# FoamUtils

A python package for OpenFOAM related tasks. WIP.

## Installation
- Install using *pip*
    ```console
    pip install git+https://github.com/johanespelund/FoamUtils.git
    ```
- Depends on PyFoam for parsing. PyFoam is a required dependency.

## Installing with uv
- Use a compatible Python version (3.10-3.12):
    ```console
    uv python pin 3.12
    ```
- Add the package:
    ```console
    uv add git+https://github.com/johanespelund/FoamUtils.git
    ```

If your environment is already created with another Python version, recreate the
virtual environment with a compatible interpreter before adding the package.
